## Install
1. Download from [Release](https://github.com/AsutoraGG/Rust-TeamChat-CLI/releases/tag/Latest)
2. Run The ``install modules.bat``
3. Open Rust and Connect Server then Open ``Get PlayerToken.bat`` and Pairing Server
4. Open ``Run Main.bat``
